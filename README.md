## Get Started

> [!Important]  
> It is recommended to use `bun` package manager for this project. This is to prevent version mismatches on development.

- Clone the repository
- Install dependencies
  - `bun install`
- Open development server:
  - `bun run dev`
- Build:
  - `bun run build`
