## Install

```bash
# NPM
npm install @vue-material/core @iconify/vue

# Yarn
yarn add @vue-material/core @iconify/vue

# PNPM
pnpm add @vue-material/core @iconify/vue

# BUN
bun install @vue-material/core @iconify/vue
```
