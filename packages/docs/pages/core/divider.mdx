## Props

### `Divider`

- `label?`: string
  - The label of the divider, overrides the direction to `x` if set
  - Default: `undefined`
- `direction?`: 'x' | 'y'
  - The direction of the divider
  - Default: `'x'`
- `size?`: SizesString
  - The size of the divider, width if direction is `x`, height if direction is `y`
  - Default: `'100%'`
- `margin?`: SizesString
  - The margin of the divider
  - Default: `'auto'`

### `WavyDivider`

- > **Inherits** `BoxProps`
- `color?`: SizesString
  - The color of the wavy divider
  - Default: `'$surface-container-highest'`
