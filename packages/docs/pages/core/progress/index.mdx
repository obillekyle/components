## Props

### `LinearProgress`

- `value?`: number
  - The value of the progress, set to a number between 0 and 100
  - Setting it to `Infinity` or a non finite number will make the progress infinite
  - Default: `Infinity`
- `size?`: number | string
  - The height of the progress bar
  - Default: `4`

### `CircularProgress`

- `value?`: number
  - The value of the progress, set to a number between 0 and 100
  - Setting it to `Infinity` or a non finite number will make the progress infinite
  - Default: `Infinity`
- `rotate?`: boolean
  - Whether to rotate the progress when it's deterministic
  - Default: `false`
- `diameter?`: number
  - The diameter of the circular progress
  - Default: `48`
- `stroke?`: number
  - The width of the circular progress
  - Default: `5`
