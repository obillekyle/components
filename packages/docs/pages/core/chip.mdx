## Props

> #### Inherits `BoxProps`, `vue.BtnHTMLAttributes`

`variant?`: filled | outlined | text | tonal | elevated

- The variant of the chip
- Default: `"filled"`

`left-icon?`: string | vue.Component

- The component that will be used to render the left icon
- Default: `undefined`

`right-icon?`: string | vue.Component

- The component that will be used to render the right icon
- Default: `undefined`

`label?`: string

- The label of the chip, can be overridden by the `slot:default`
- Default: `undefined`
