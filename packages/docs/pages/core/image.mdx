## Props

### `BlockImage`

- > **Inherits `BoxProps`**
- `src?`: string | Blob
  - The source of the image
  - Default: `undefined`
- `alt?`: string
  - The alt of the image
  - Default: `undefined`
- `width?`: number
  - css.width
  - the image's width
  - Default: `undefined`
- `height?`: number
  - css.height
  - The image's height
  - Default: `undefined`
- `lazy?`: boolean
  - Whether to lazy load the image when it's visible on the screen
  - Default: `false`
- `fit?`: 'contain' | 'cover' | 'fill'
  - css.object-fit
  - How to fit the image to the frame
  - Default: `'cover'`
- `position?`: 'left' | 'center' | 'right' | 'top' | 'bottom'
  - css.object-position
  - The position of the image in the frame
  - Default: `'center'`
- `span?`: boolean
  - Whether to span the frame to the parent element
  - Default: `false`
- `ratio?`: number
  - css.aspect-ratio
  - The ratio of the image
  - Default: `undefined`

### `SquareImage`

- > **Inherits `BoxProps`**
- `src?`: string | Blob
  - The source of the image
  - Default: `undefined`
- `size?`: number
  - the image's height and width
  - Default: `96`
- `lazy?`: boolean
  - Whether to lazy load the image when it's visible on the screen
  - Default: `false`
- `frame?`: 'circle' | 'clover' | 'hexagon' | 'default'
  - The shape of the image
  - Default: `'default'`
- `radius?`: number
  - The radius of the image frame. Only works when `frame` is `'default'`
  - Default: `'#xs'`
