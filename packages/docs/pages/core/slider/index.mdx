## Props

### `Slider`

- `default-value?`: number
  - The default value of the slider
  - Default: `props.min` | `props.values.sort()[0]` | `0`
- `values?`: number[] | \{label: string, value: number}[]
  - The values of the slider, overwrites `min`, `max` and `step` if defined
  - Default: `undefined`
- `min?`: number
  - The minimum value of the slider
  - Default: `0`
- `max?`: number
  - The maximum value of the slider
  - Default: `100`
- `step?`: number
  - The step of the slider
  - Default: `1`
- `decimal?`: number'
  - The number of decimal places to show
  - Default: `0`
- `show-value?`: boolean
  - Whether to show the value of the slider
  - Default: `false`
- `show-label?`: boolean
  - Whether to show the value labels of the slider, only if `values` is defined
  - Default: `false`
- `v-model:modelValue`: number
  - The value of the slider
  - Default: `undefined`
- `change?`: (value: number) => void
  - The callback when the value of the slider changes
  - Default: `undefined`

### `BlockSlider`

- `default-value?`: number

  - The default value of the slider
  - Default: `props.min` | `0`

- `min?`: number

  - The minimum value of the slider
  - Default: `0`

- `max?`: number

  - The maximum value of the slider
  - Default: `100`

- `step?`: number

- `decimal?`: number

- `icon?`: string | vue.Component

- `v-model:modelValue`: number

  - The value of the slider
  - Default: `undefined`

- `change?`: (value: number) => void
  - The callback when the value of the slider changes
  - Default: `undefined`
