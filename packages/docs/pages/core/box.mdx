## Wrapping custom components

<Preview>
  <CodeHighlight>
    ```vue
    ./my-custom-panel.vue

    <script setup lang="ts">
      import { Box, getBoxProps, type BoxProps } from '@vue-material/core'

      // Extending using BoxProps
      interface PanelProps extends BoxProps {
        title: string
      }

      const props = defineProps<PanelProps>()
      const boxProps = getBoxProps(props, {
        // add default box props here or use vue.withDefaults
        as: 'header'
        height: '#component-lg'
      })
    </script>

    <template>
      <Box
        v-bind="boxProps"
        class="my-custom-panel"
      >
        <h1>{{ props.title }}</h1>
      </Box>
    </template>

    <style>
      /* Custom styles */
    </style>
    ```

  </CodeHighlight>
</Preview>

## Props

#### Interface `BoxProps`

`as?`: string | vue.Component
- The component that will be used to render the box
- Default: `"div"`

`p?`: SizesString
- css.padding
- The padding of the box
- Default: `undefined`
- **Subprops**:
  - `px?`: SizesString - css.padding-inline
  - `py?`: SizesString - css.padding-block
  - `pl?`: SizesString - css.padding-left
  - `pr?`: SizesString - css.padding-right
  - `pt?`: SizesString - css.padding-top
  - `pb?`: SizesString - css.padding-bottom

`m?`: SizesString
- css.margin
- The margin of the box
- Default: `undefined`
- **Subprops**:
  - `mx?`: SizesString - css.margin-inline
  - `my?`: SizesString - css.margin-block
  - `ml?`: SizesString - css.margin-left
  - `mr?`: SizesString - css.margin-right
  - `mt?`: SizesString - css.margin-top
  - `mb?`: SizesString - css.margin-bottom

`r?`: SizesString
- **css.border-radius**
- The border radius of the box
- Default: `undefined`

`width?`: SizesString
- css.width
- The width of the box
- Default: `auto`

`height?`: SizesString
- css.height
- The height of the box
- Default: `auto`

~~`@deprecated bg?`: ColorVariants | 'on-bg'~~
- The color theme of the box
- Default: `undefined`

~~`@deprecated exclude?`: boolean~~
- Remove the property `md-box` from the component
  - Default: `false`
