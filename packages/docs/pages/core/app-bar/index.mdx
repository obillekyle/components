## Headline

The `Headline` component can be used to set the title of the AppBar.
Make sure to that the component is nested inside of the `Layout`
components' `default` slot.

<Preview>
  <CodeHighlight>
    ```vue
    <Headline title="Title" />
    ```
  </CodeHighlight>
</Preview>

## Props

### Headline

- > **Inherits `BoxProps`**
- `title?`: string
  - The title that will be displayed in the AppBar
  - Default: `undefined`

### TopAppBar

- `#actions`: slot
  - The actions that will be displayed in the right side of the AppBar
