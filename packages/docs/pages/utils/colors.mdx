**`static from()`**: Colors

- Create a new color object from a string or an existing color object
- Parameters
  - `color`: string | String | Colors

**`static isLight()`**: boolean

- Check if a color is light
- Parameters
  - `color`: string

**`constructor()`**: Colors

- Creates an instance of Colors
- Parameters
  - `color`: string | String
    - Default: `'white'`
  - `curve`: BezierEasing
    - Bezier easing function
    - Default: `BezierEasing([0.2, 0.2, 1, 1])`

**`set()`**: Colors

- Updates and returns the Colors object
- Parameters
  - `colorString`: string | String

**`shade()`**: string

- Get a shade of the current color
- Parameters
  - `shade`: number
  - `alpha?`: number
    - Default: `1`

**`isLight()`**: string

- Check if the current color is light
- Parameters
  - `shade?`: number
    - Default: `50`

**`get opposite`**: Colors

- Get the opposite of the current color

**`get hex`**: string

- Get the hex value of the current color

**`toString()`**: string

- Get the string representation of the current color
