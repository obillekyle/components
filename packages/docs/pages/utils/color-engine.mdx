### Properties and Methods

**`colors`: ColorObject**

- Stores all the colors passed on the constructor

**`constructor(colors)`: ColorEngine**

- Creates an instance of ColorEngine
- Parameters
  - `color?`: string | Partial\<ColorOptions\<string | Colors>>
    - Default: `'white'`

**`getColorVariables(theme)`: ColorVariables**

- generate a color pallete based on the MD3's color system guidelines
- Parameters
  - `theme?`: 'light' | 'dark'
    - Specify which theme to use when generating the pallete
    - Default: `'light'`

**`getShades(theme)`: Record\<string, string>**

- generate a shades object
- Format `<color-key>-<shade> | <color-key>-<shade>-<opacity>`
- Parameters
  - `theme?`: 'light' | 'dark'
    - Specify which theme to use when generating the shades
    - Reverses the shades when dark, so shade 20 in light theme is 80 in dark theme
    - Default: `'light'`
